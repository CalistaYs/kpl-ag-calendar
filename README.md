# KPL Chengdu AG Calendar

This repository publishes a subscribable calendar for Chengdu AG KPL matches.

## iOS subscription URL

Paste this URL into iPhone Settings > Calendar > Accounts > Add Account > Other > Add Subscribed Calendar:

```text
https://raw.githubusercontent.com/CalistaYs/kpl-ag-calendar/main/calendar.ics
```

If iOS asks for a server, use the same URL above.

## Updates

The calendar file can be updated in this repository and the iOS subscription will keep using the same URL.
